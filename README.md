# Redact Tool Plugin for Obsidian.md.

This plugin adds a new configurable command to redact the selected text.

Features:
- Ignores control characters like tabs and new lines.
- Override for the redaction character. (Default character is "█")
- Toggle for ignoring spaces. (Set to true by default)
- Toggle for ignoring symbols. (Set to true by default)

---

I made this plugin primarily for personal use, I may fix problems here and there but don't expect consistent updates.
